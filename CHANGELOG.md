# Changelog

All notable updates to the houserules are recorded here.

## 2026-07-05

- Added the *Caravan Trade Subsystem* in `subsystems/`, including rules for
  caravan stock, working capital, mediated orders, trade journeys, route
  demand, recurring merchants, and campaign-ready Otari caravan examples.
- Expanded the caravan rules with perk progressions, VTT-ready budget formulas,
  stronger item-access guardrails, and trade friction that better matches both
  Pathfinder 2e balance and a high-magic caravan economy.
- Added the *Drugs Subsystem* in `subsystems/drugs-making.md`, transcribed from
  the Notion source into Markdown so the original rules, actions, tags, and
  component table are preserved in repository form.
- Cleaned up the drugs documentation so the Markdown uses consistent headings,
  tag formatting, and table text while keeping the rules content intact.
- Rewrote the project introduction so `README.md` presents *Pathfinder 2e
  Reforged* as Apostol Apostolov's house-rule framework rather than just a file
  index.

## 2024-12-16

- Clarified the opening guidance for the remaster rules so the relevant DC
  choices are easier to interpret at the table.
- Tightened several feat entries, including small wording fixes that make the
  affected options easier to apply consistently.
- Folded in a few late-2024 errata points so the house rules stay aligned with
  the newer remaster assumptions.

## 2024-12-03

- Clarified how retry options work for the recall-knowledge style rules, so
  players can see when a second attempt is allowed and when the difficulty
  increases instead.

## 2024-11-24

- Added a clearer explanation of how auras and emanations behave when walls,
  doors, and other blockers are involved.
- Continued the barbarian feat revisions by folding the charge-related options
  into cleaner, easier-to-read text.
- Adjusted the death and massive-damage rules so they are less punishing in the
  edge cases the table was running into most often.

## 2024-11-23

- Applied general wording cleanup across the main rules pages, mostly to reduce
  ambiguity and improve table usability.

## 2024-08-09

- Added the subtle spellcasting feat chain to the feats chapter so the
  concealment and silence options sit together in one place.
- Improved the crafting and downtime references so they read more like usable
  procedures and less like scattered notes.

## 2024-08-08

- Added another feat entry to the consolidated feats chapter and adjusted the
  surrounding layout so later feat additions stay easy to scan.

## 2024-08-05

- Clarified how Aid works when a creature is distracted or otherwise not fully
  focused on the scene.
- Cleaned up several core action descriptions so the action timing and
  interaction limits are easier to follow.

## 2024-08-03

- Added new basic actions to support edge cases the core rules did not cover
  cleanly.
- Reorganized the feats material into a dedicated page so feat updates no
  longer have to be scattered across multiple sections.

## 2024-08-02

- Expanded stealth and detection clarifications, especially around distracted
  creatures, ambushes, and when initiative should shift.
- Added the earlier companion and support rules that help resolve familiar and
  ambush handling without improvised table rulings.

## 2024-07-29

- Improved crafting and gear handling, including the parts of the system that
  affect item prep, carry limits, and practical equipment use.
- Added the lift and ally-passing actions to cover table situations that the
  core action list handled awkwardly or not at all.

## 2024-07-17

- Refined the familiar reach-mode rules so the distance bands and combat
  expectations are clearer.

## 2024-04-04

- Reworked the wording of the recall-knowledge success tables to make the
  degrees of success easier to read and apply during play.

## 2023-12-13

- Added a clarification that stops item swapping from being treated as a
  single Interact action.

## 2023-12-02

- Added the basic action entries needed for lifting objects and passing items
  or signals in play.

## 2023-11-30

- Added a downtime activity for building friendships and local contacts, so
  social investment has a defined mechanical path.

## 2023-11-29

  workspace so the related material could be maintained alongside the main
  houserules.

## 2023-11-26

  activity is easier to apply in the field.

## 2023-11-25

- Began the spells chapter and started collecting the spellcasting rules that

## 2023-11-23

- Clarified the restrained and related combat rules so the action economy and

## 2023-11-21

- Updated the combat-facing spell and condition rules, mostly around how

## 2023-11-20

- Moved some specialized material into dedicated reference pages so the main
