# Pathfinder Arena

## Basic Rules

- Arena teams are built from common creatures from first-party Pathfinder 2e
  sources.
- Each creature has an XP cost based on the event level.
- Every creature on a team must share at least one keyword with the others.
- Creatures begin with weapons drawn and ammunition ready.

## Event XP Budget

| Event Type | Budget XP |
| --- | ---: |
| Village Pit Fights | 100 |
| Standard Coliseum | 160 |
| Grand Coliseum | 240 |

## Creature XP Costs

| Creature Level | XP |
| --- | ---: |
| Event Level - 2 | 20 |
| Event Level - 1 | 30 |
| Event Level | 40 |
| Event Level + 1 | 60 |
| Event Level + 2 | 80 |
| Event Level + 3 | 120 |

## Creature Restrictions

- Ethereal or similarly untouchable creatures are excluded.
- Extremely safe creatures or ones that can vanish beyond reasonable combat
  reach are excluded.

## Hero Points

- Each team gains a Hero Point at the start of the first three rounds.
- Hero Points are only for rerolling attacks and skill checks.

## Creature Advancement

- Death is final in lethal arena fights.
- Survivors can become veterans and later elites after enough fights.
- Veteran and elite titles are part of the arena presentation.

## Team Fund

- Some events grant a gold budget for equipment.
- Organizers can scale the event level up or down, but do so evenly.

## Game Modes

- Standard: normal arena deathmatch.
- Tag Team: paired teams rotate in sequence.
- Party Clash: player characters replace monsters.
- Leader: one player character can lead the team.
- Massacre: multiple team waves are released over time.
- Clown Defense: a support clown can generate Hero Points for the team.
- King of the Hill: control the top platform for several turns.
- Monster Bash: both teams fight a released monster.
- Draft: teams draft from shared creature pools.

## Appendix

- Average damage tables help judge fair arena matchups.
- Teams are assumed to know the opposing roster's abilities.
